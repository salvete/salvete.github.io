声明Bean类

对于每个需要加入到Spring容器中的类，使用@Component注解进行声明。其派生出三个注解：

  • Dao层：@Repository
  • Service层：@Service
  • Controller层：@Controller

其中，如果类中属性为普通类型，则使用@Value即可进行赋值；如果类中属性为自定义类，则可以使用@Autowired，若@Configuration中超过一个同类型的Bean，则需要使用@Qualifier显式地指定。示例代码如下：
