第一章内容

书的第一章介绍了一些基本概念，结合书上的例子了解即可，基本概念如下：

  • atom
  • S-expression
  • list

然后介绍了几种运算，很好理解：

  • car: 返回非空列表的第一个元素，如(car ‘(a b c d)) 为a
  • cdr:返回非空列表除去第一个元素之后的剩下元素组成的列表。注意，返回值为列表。可以看到，使用cdr可以使问题的规模减1，这为递归提供了工具
  • cons:将元素加入到列表表头，如(cons ‘a ‘(b c d e))为(a b c d e)
  • null?:判断列表是否为空，判断递归边界时使用
  • eq?:判断两个原子是否相等

关于scheme

这是一个关于scheme的教程：Scheme 语言简明教程

由于刚刚接触scheme，对括号的使用不熟悉，导致了运行错误。这里是括号的使用方法，可能对你有帮助：使用括号的时候，系统会默认第一个符号为调用。所以，以下的表达式是合法的：

  • (+ 1 2)，+也是一个调用
  • (car ‘(a b c)),car是一个调用
  • (cdr ‘(a b c)),cdr是一个调用
  • ……

但是下面括号的使用是非法的：

  • (1),1本身不是一个调用
  • (‘a),a本身不是一个调用
  • ((car ‘(a b c))),因为car ‘(a b c))返回的是a,(a)不是一个合法的使用
  • (((((1))))),这个大错特错了，因为首先(1)错了，更不用说外面那么多层括号
  • ……

总结

刚开始需要熟悉的是scheme的语法，熟悉之后就好办了。
